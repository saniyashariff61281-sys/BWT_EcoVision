BWT_EcoVision – The Invisible Energy Drain

Theme
Climate Tech Engines 

Problem Statement
Energy is wasted every day in homes and offices because no one notices it.  
Examples: Lights left ON, AC running unnecessarily, devices in standby.  
This leads to unnecessary energy consumption and increased carbon footprint.

Proposed Solution
EcoWatch is a smart monitoring system that makes invisible energy waste visible.  
It calculates energy usage of devices and suggests ways to reduce waste.  
It also estimates potential CO₂ reduction if optimizations are followed.

Features
- Track energy consumption of home/office devices
- Alert if energy is being wasted
- Show estimated carbon reduction
- Simple dashboard for suggestions

Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Database: Optional (CSV or JSON to store user input)

The system works as follows:
1. **User** inputs devices and usage hours  
2. **Frontend** sends data to backend  
3. **Backend/AI Engine** calculates energy waste and carbon footprint  
4. **Backend** sends suggestions and stats  
5. **Frontend** displays results and graph/dashboard
